| :--- | :--- | :--- |
| `Coding & Tech` | `/debug`, `/refactor`, `/optimizecode`, `/security` | Controle técnico direto |
| `Output Modes` | `/ghost` (só código), `/stepbystep`, `/checklist` | Formato da resposta |
| `Thinking Styles` | `/critic`, `/analyst`, `/firstprinciples`, `/simplify` | Personalidade da IA |
| `Advanced Control` | `/toneformal`, `/bullet`, `/limit`, `/rewrite` | Ajuste fino de saída |

💡 **Dica:** Combine `@skill` + `/comando` para controle total. Ex: `"Use @frontend-design + /critic para revisar o layout antes de gerar o código final."`
